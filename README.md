# Claude Code 테스트

## 프로젝트 소개

디저트 원가 계산기는 제과제빵 사업자를 위한 웹 기반 원가 관리 도구입니다. 재료 재고를 관리하고, 레시피를 작성하며, 정확한 원가와 판매가를 계산할 수 있습니다.

## 주요 기능

- 📦 **재고 관리**: 재료별 단가 자동 계산 및 재고 DB 관리
- 📝 **레시피 작성**: 재료 선택 및 사용량 입력으로 간편한 레시피 작성
- 💰 **원가 계산**: 1회 제작 원가, 생산 수량별 총 비용, 마진율 적용 판매가 자동 계산
- 📋 **레시피 목록**: 저장된 모든 레시피 관리 및 불러오기
- 📤 **데이터 내보내기**: JSON 형식으로 레시피 백업 및 공유
- 📥 **데이터 가져오기**: 저장된 레시피 파일 불러오기
- 🖨️ **HTML 출력**: 레시피별 원가 계산서 다운로드 (인쇄 가능)

## 기술 스택

- React 18
- Tailwind CSS
- LocalStorage (데이터 저장)
- 바닐라 JavaScript (Babel로 JSX 변환)

## 사용법

### 1. 프로젝트 실행

이 프로젝트는 별도의 설치나 빌드 과정 없이 바로 실행할 수 있습니다.

```bash
# 간단한 HTTP 서버로 실행 (Python 3)
python -m http.server 8000

# 또는 Node.js의 http-server 사용
npx http-server
```

브라우저에서 `http://localhost:8000`으로 접속하세요.

### 2. 재고 등록

1. **재고 관리** 탭으로 이동
2. 재료명, 단위, 총 수량, 총 비용 입력
3. **추가** 버튼 클릭
4. 단가가 자동으로 계산되어 저장됩니다

### 3. 레시피 작성

1. **레시피 작성** 탭으로 이동
2. 레시피 이름 입력
3. 재료 추가 방식 선택:
   - **재고에서 선택**: 등록된 재고에서 재료 검색 및 선택 (비용 자동 계산)
   - **직접 입력**: 새로운 재료 직접 입력
4. 생산 수량, 포장재 비용, 마진율 입력 (선택사항)
5. **레시피 저장하기** 클릭

### 4. 레시피 관리

1. **목록** 탭에서 저장된 레시피 확인
2. 각 레시피에서 다음 작업 가능:
   - **불러오기**: 레시피 편집
   - **HTML**: 원가 계산서 다운로드
   - **삭제**: 레시피 제거

### 5. 데이터 백업

- **내보내기**: 선택한 레시피를 JSON 파일로 저장
- **가져오기**: 저장된 JSON 파일을 불러와 레시피 복원

## 데이터 저장

모든 데이터는 브라우저의 LocalStorage에 저장됩니다. 브라우저 캐시를 삭제하면 데이터가 손실될 수 있으므로 정기적으로 백업하시기 바랍니다.
